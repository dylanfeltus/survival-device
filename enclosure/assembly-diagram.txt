╔════════════════════════════════════════════════════════════════════════════╗
║                          ASSEMBLY SEQUENCE                                 ║
╚════════════════════════════════════════════════════════════════════════════╝

    STEP 1: Prepare Bottom Shell
            ↓
            Install 4× M3 heat-set inserts in corner bosses
            Use soldering iron at 220°C (PLA) or 250°C (PETG)

    STEP 2: Install Batteries
            ↓
            Place 2× 18650 batteries in compartment (side-by-side)
            Ensure polarity matches battery holder markings

    STEP 3: Mount UPS HAT B
            ↓
            Connect to battery holder terminals
            Secure to standoffs with M2.5 screws

    STEP 4: Install Orange Pi Zero 2W
            ↓
            Connect to UPS HAT via GPIO/pogo pins
            Align ports with side cutouts
            Secure with M2.5 screws

    STEP 5: Connect LCD HAT
            ↓
            Attach to Orange Pi 26-pin GPIO header
            Verify display window alignment
            Secure to top standoffs with M2.5 screws

    STEP 6: Mount Rotary Encoder (in Top Shell)
            ↓
            Thread shaft through hole
            Secure body with hot glue or tape
            Attach knob to shaft

    STEP 7: Close Enclosure
            ↓
            Route wires carefully (avoid pinching)
            Align top shell with alignment pins
            Insert 4× M3 screws and tighten diagonally

    STEP 8: Final Testing
            ↓
            Connect USB-C power
            Verify display, ports, and encoder function
            Thermal test (30 minutes runtime)


╔════════════════════════════════════════════════════════════════════════════╗
║                         TROUBLESHOOTING GUIDE                              ║
╚════════════════════════════════════════════════════════════════════════════╝

    ISSUE: Case won't close
    FIX:   ├─ Check for wire pinching at shell interface
           ├─ Verify all components are seated properly
           └─ Check for warped print (reheat and flatten if needed)

    ISSUE: LCD doesn't align with window
    FIX:   ├─ Loosen PCB mounting screws and adjust position
           ├─ Add shims under PCB standoffs
           └─ Modify .scad file `lcd_active_width/height` variables

    ISSUE: Ports not accessible
    FIX:   ├─ Verify Orange Pi is oriented correctly (180° check)
           ├─ File/enlarge port cutouts if needed
           └─ Check PCB standoff heights

    ISSUE: Overheating
    FIX:   ├─ Verify ventilation slots are clear
           ├─ Add heatsink to Orange Pi SoC chip
           └─ Reduce CPU governor to powersave mode

    ISSUE: Screws won't tighten (stripped threads)
    FIX:   ├─ Use heat-set inserts (essential for reusability)
           ├─ Don't overtighten — plastic has limits
           └─ If stripped, fill hole with epoxy and re-drill
